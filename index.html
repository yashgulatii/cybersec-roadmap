Phase 1 — Security+ SY0-701
Weeks 7–18 · Professor Messer free videos · All 5 domains
0%
Primary resource: All videos at professormesser.com/sy0-701 — 100% free. Watch one section, take notes in your own words, then do practice questions before moving on. Don't binge-watch.
Domain 1: General Security Concepts (12% of exam)
CIA Triad: Confidentiality, Integrity, Availability
With real examples of each being violated. Non-repudiation concept.
Exam
Authentication factors: something you know/have/are + location/action
MFA, passwordless auth, biometrics. Understand AAA framework (Authentication, Authorization, Accounting).
Exam
Cryptography: symmetric vs asymmetric, key exchange
AES, 3DES, RSA, ECC. Know key lengths and use cases. Diffie-Hellman key exchange.
Exam
Hashing algorithms: MD5, SHA-1, SHA-256, SHA-3
Why MD5/SHA-1 are deprecated. Understand collisions, salting, HMAC.
Exam
PKI: Certificates, CA hierarchy, CRL, OCSP, certificate types
How HTTPS works end-to-end. DV vs OV vs EV certificates. Self-signed vs CA-signed.
Exam
Security controls: preventive, detective, corrective, deterrent, compensating
Technical vs managerial vs operational controls. Physical controls.
Exam
Zero Trust model: never trust, always verify, least privilege
Microsegmentation, identity verification, continuous validation. Compared to perimeter security.
Exam
Domain 2: Threats, Vulnerabilities & Mitigations (22% — heaviest weight)
Social engineering: phishing, vishing, smishing, spear phishing, whaling, pretexting, baiting
Identify each type. Understand why humans are the weakest link. Defence strategies.
Exam
Malware types: virus, worm, trojan, ransomware, rootkit, spyware, adware, keylogger, botnet, RAT
Understand propagation method + payload for each. Fileless malware, living-off-the-land (LoTL).
Exam
Attack types: DoS, DDoS, on-path (MITM), replay, spoofing, buffer overflow
Mechanisms and mitigations for each. Amplification attacks (DNS, NTP).
Exam
Web application attacks: XSS, SQL injection, CSRF, directory traversal, command injection
How each works at HTTP level. OWASP Top 10. Mitigations: input validation, WAF, CSP.
Exam
Password attacks: brute force, dictionary, rainbow table, credential stuffing, spraying
Salting/hashing defences. Account lockout policies. Why password managers matter.
Exam
Vulnerability scanning: active vs passive, credentialed vs non-credentialed, false positive/negative
CVSS score components (AV, AC, PR, UI, S, C, I, A). CVE/NVD databases.
Exam
Threat intelligence: OSINT, dark web, IOCs, threat feeds, STIX/TAXII
Indicators of Compromise vs Indicators of Attack. Threat actors: nation-state, criminal, hacktivist, insider.
Exam
Supply chain attacks, zero-day exploits, APTs
SolarWinds-style attack as a case study. What makes APTs different from script kiddies.
Exam
Domain 3: Security Architecture (18%)
Network segmentation: VLANs, DMZ, microsegmentation, screened subnets
Draw a typical network diagram with DMZ. Know where each device type lives.
Exam
Firewalls: stateful vs stateless, NGFW, WAF, ACL rules
Packet filtering vs application layer inspection. Implicit deny. Rule ordering.
Exam
IDS vs IPS: signature-based, anomaly-based, inline vs passive
Snort rules basics. Understanding alerts vs blocks. HIDS vs NIDS.
Exam
VPN: IPsec vs SSL/TLS, site-to-site vs remote access, split tunnelling
IKE phases, tunnel vs transport mode. SD-WAN mention.
Exam
Cloud security: IaaS/PaaS/SaaS shared responsibility model
AWS/Azure/GCP security basics. Cloud misconfigurations (S3 bucket exposure). CASB.
Exam
Secure network design: defence in depth, least privilege, fail-open vs fail-closed
Understand why layered security beats single-point defences.
Exam
Identity & Access Management: LDAP, Active Directory, SSO, SAML, OAuth, OIDC
Federation, role-based access control (RBAC), attribute-based (ABAC), mandatory (MAC), discretionary (DAC).
Exam
Domain 4: Security Operations (28% — highest weight, most SOC-relevant)
Incident response lifecycle: Preparation → Detection → Analysis → Containment → Eradication → Recovery → Lessons Learned
NIST SP 800-61 framework. Know each phase's actions and who is responsible.
Exam + SOC
Log types: Windows Event Logs, Syslog, Web server logs, firewall logs, auth logs
Know key Windows Event IDs: 4624 (login), 4625 (failed login), 4648, 4720, 7045. Read raw logs.
Exam + SOC
SIEM: log aggregation, correlation rules, alerts, dashboards
What SIEM does and doesn't do. Integration with threat intel. SOAR automation.
Exam + SOC
Endpoint security: AV, EDR, DLP, application whitelisting, host-based firewall
EDR vs traditional AV. Understand telemetry-based detection vs signature detection.
Exam
Vulnerability management lifecycle: discover, prioritize, remediate, verify
Patch management, rollback, compensating controls when patching isn't possible.
Exam
Digital forensics basics: chain of custody, order of volatility, imaging drives
What to collect first in an incident (RAM before disk). Legal considerations. Write-blocking.
Exam + SOC
Threat hunting: hypothesis-driven investigation, IOC vs TTP-based hunting
Difference from incident response — proactive vs reactive. MITRE ATT&CK as hunting framework.
Exam + SOC
Network monitoring: NetFlow, packet capture, IDS alerts, DNS sinkholes
East-west vs north-south traffic. Baseline normal to detect anomalies.
Exam + SOC
Domain 5: Security Program Management (14%)
Risk management: threat × vulnerability × impact, qualitative vs quantitative
Risk matrix, residual risk, risk acceptance/avoidance/transference/mitigation.
Exam
Compliance frameworks: NIST CSF, ISO 27001, SOC 2, PCI-DSS, HIPAA, GDPR
Know what each covers and which industries they apply to. You won't be tested on implementation details.
Exam
Data classification: public, private, sensitive, confidential, PII, PHI
Data retention, destruction policies, DLP controls per classification level.
Exam
BCP/DR: RTO, RPO, hot/warm/cold sites, backup strategies, tabletop exercises
Recovery Time Objective vs Recovery Point Objective. What each site type means.
Exam
